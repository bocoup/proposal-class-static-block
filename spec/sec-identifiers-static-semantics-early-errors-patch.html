Static Semantics: Early Errors

BindingIdentifier : Identifier
  • It is a Syntax Error if the code matched by this production is contained in strict mode code and the StringValue of |Identifier| is *"arguments"* or *"eval"*.
  • It is a Syntax Error if the code matched by this production is nested, directly or indirectly (but not crossing function boundaries), within a |ClassStaticBlock| and the StringValue of |Identifier| is *"await"*.
LabelIdentifier : Identifier
  • It is a Syntax Error if the code matched by this production is nested, directly or indirectly (but not crossing function boundaries), within a |ClassStaticBlock| and the StringValue of |Identifier| is *"await"*.
IdentifierReference : Identifier
  • It is a Syntax Error if the code matched by this production is nested, directly or indirectly (but not crossing function boundaries), within a |ClassStaticBlock| and the StringValue of |Identifier| is *"arguments"* or *"await"*.
